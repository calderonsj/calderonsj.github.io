¿Cómo funciona un CFD?

Primeros pasos

¿Dónde está OpenFOAM?

/opt/OpenFOAM/

¿De qué se compone?

  • Volvamos a nuestro home. Teclea blockMesh. ¿Funciona?
  • Vamos a crear un directorio de trabajo:
    • mkdir -p $FOAM_RUN
  • Vamos a trabajar con nuestro primer tutorial. Lo copiamos a nuestro directorio recién creado.
    • cd $FOAM_RUN
    cp -r $FOAM_TUTORIALS/incompressible/icoFoam/cavity/cavity .

OpenFOAM Set Up

source ~/opt/OpenFOAM-x.x/etc/bashrc

Tendríamos que repetir esto cada vez que abrimos terminal. Un poco rollo, ¿no?

  • Mejor abrimos desde home el archivo ./bashrc. ¿Recuerdas cómo verlo? ls -
  • Una vez abierto (mousepad), vamos a la última línea. Incluimos la linea de código ahí
  • Hay una forma aún mejor. Con un alias. Esto es útil si tenemos varias versiones de OF.
    • source ofxx='~/opt/OpenFOAM-x.x/etc/bashrc'
  • En la terminal, para cargar OF habrá que teclear el alias de la versión que queramos

Variables de entorno

  • A veces, es útil definir variables de entorno para facilitar el acceso a ciertas rutas, o almacenar variables, etc.
  • OpenFOAM tiene algunas de ellas. Normalmente empiezan con WM_ o FOAM_
  • ¿Qué acciones podemos realizar?
    • echo $MY_ENV_VAR
    ls -l $MY_ENV_VAR
    cd $MY_ENV_VAR
