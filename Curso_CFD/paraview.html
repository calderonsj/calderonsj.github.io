Tutorial ParaView

v5.0.1

Interfaz

  • Barra de menú
  • Barra de herramientas
  • Pipeline browser
  • Barra de propiedades
  • Visor 3D
  1. Barra de menú: permite acceder a la mayoría de características del programa, como herramientas de archivo, edición, etc.
  2. Barra de herramientas: permite el acceso rápido a las herramientas más empleadas en paraView
  3. Pipeline browser: paraView organiza los datos y las acciones a realizar sobre ellos con este tipo de arquitectura. Te permite ver la secuencia de operaciones y acciones realizadas, así como los datos cargados al programa de forma estructurada.
  4. Barra de propiedades: permite ver y cambiar los parámetros asociados a una acción particular.
  5. Visor 3D: la interfaz propiamente dicha. Permite visualizar los datos.

Streamlines

  • Selecciona el filtro Stream Tracer
  • Seed type: High Resolution Line Source. Eje Y
  • Resolution: 20

Pipeline

  • Sobre el streamline creado:
  • Filter: tube
  • Radio

Contour

  • Selecciona el filtro Contour
  • Contour by: p field
  • Establece el rango en isosurfaces (add range)
  • Para obtener un contour en 2D, es necesario aplicar un filtro Slice antes (lo vemos más adelante).

Velocity vectors

  • Selecciona el filtro Glyph
  • Glyph type: arrow
  • Active Attributes: vectors: U

Calculator

  • Para calcular nuevos campos
  • Realiza operaciones matemáticas sobre los campos existentes

Clip y slice

  • Clip:corta parte del dominio
  • Slice: corta un plano del dominio

  • Clip:corta parte del dominio

Plot over line

  • Selecciona el filtro Plot Over Line
  • En la barra de propiedades, selecciona los puntos inicial y final de la línea
  • En Series Parameters, puedes seleccionar los campos que representar

Guardar una imagen

  • En la barra de menú, selecciona File -> Save Screenshot
  • Selecciona la resolución.
  • Selecciona la localización para guardar la imagen, y su formato (PNG, JPEG,etc.)

Crear una animación

  • En la barra de menú, selecciona File -> Save Animation
  • Selecciona los fps (24 para tiempo real) y la resolución
  • Selecciona la localización para guardar la animación, y su formato (conjunto de frames en PNG o JPEG, o formato de vídeo .ogv)

Si guardas como un conjunto de frames en, por ejemplo, JPEG, para crear un video, desde la terminal tienes que introducir el comando:

ffmpeg -i cavity.000%d.jpg video.mp4

Cambiar formato input de acuerdo a tu salida de paraView.

Otros filtros

  • Threshold: selecciona un conjunto de celdas de acuerdo a un rango (campo escalar)
  • AnnotateTimeFilter: introduce un texto con el tiempo de simulación. Se puede formatear para escalarlo, adimensionalizarlo, introducir texto, etc.
  • Extract Selection: extrae un conjunto de celdas seleccionadas para considerarlas como un subconjunto de datos sobre el que aplicar acciones.
  • Reflect: copia la geometría sobre un plano de simetría.
  • Descriptive Statistics: calcula la información de los campos seleccionados y calcula variables estadísticas, como la media o la desviación estándar, para cada paso de tiempo.

Recuerda que para ver más filtros, en la barra de menú -> Filters -> Alphabetical
